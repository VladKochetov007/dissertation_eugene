# Reading Notes: Kim (2021) — Meta-MAPG

**Paper:** "A Policy Gradient Algorithm for Learning to Learn in Multiagent Reinforcement Learning"  
**Authors:** Dong-Ki Kim, Miao Liu, Matthew Riemer, Chuangchuang Sun, Marwa Abdulhai, Golnaz Habibi, Sebastian Lopez-Cot, Gerald Tesauro, Jonathan P. How  
**Venue:** ICML 2021  
**Key topic:** Meta-multiagent policy gradient theorem

---

## Key Claims

1. **Fundamental challenge in MARL:** Each agent perceives non-stationary environment due to other simultaneously learning agents
2. **Prior work limitation:** 
   - Meta-PG (Al-Shedivat 2018) only considers agent's own learning dynamics
   - LOLA (Foerster 2018) only considers shaping other agents' learning
   - Neither considers BOTH simultaneously
3. **Meta-MAPG theorem:** New policy gradient that accounts for:
   - Agent's own non-stationary policy dynamics (own learning)
   - Non-stationary policy dynamics of other agents (peer learning)
4. **Theoretical unification:** Meta-MAPG inherently comprises benefits of both Meta-PG and LOLA
5. **Empirical result:** Superior adaptation to new agents across mixed incentive, competitive, cooperative domains

---

## Mathematical Setup

### Stochastic Game
n-agent stochastic game: M_n = ⟨I, S, A, P, R, γ⟩
- I = {1,...,n} — set of n agents
- S — state space
- A = ×_{i∈I} A^i — joint action space
- P: S × A → S — transition probability
- R = ×_{i∈I} R^i — reward functions
- γ ∈ [0,1) — discount factor

Each agent i has policy π^i(a^i|s, φ^i) parameterized by φ^i.

### Markov Chain of Policies

Key insight: MARL inherent non-stationarity = Markov chain of joint policies

Initial joint policy: φ_0 = {φ^i_0, φ^{-i}_0}
↓ (collect K trajectories τ_{φ_0})
Inner-loop update: φ^i_1 ~ U^i(τ_{φ_0}, φ^i_0)
↓
Joint policy: φ_1 = {φ^i_1, φ^{-i}_1}
↓ (continue for L steps)
φ_L

**Markovian update function (inner-loop):**
```
φ^i_{ℓ+1} = U^i(τ_{φ_ℓ}, φ^i_ℓ) = φ^i_ℓ + α^i ∇_{φ^i_ℓ} E[G^i(τ_{φ_ℓ})]
```

### Meta-Optimization Objective

Optimize initial policy φ^i_0 to maximize expected adaptation performance over Markov chain:

```
max_{φ^i_0} E_{p(φ^{-i}_0)} [Σ_{ℓ=0}^{L-1} V^i_{φ_0:ℓ+1}(s_0, φ^i_0)]
```

where:
```
V^i_{φ_0:ℓ+1}(s_0, φ^i_0) = E_{p(τ_{φ_0:ℓ}|φ_0:ℓ)} E_{p(τ_{φ_ℓ+1}|φ_ℓ+1)} [G^i(τ_{φ_ℓ+1})]
```

This is the **meta-value function** — generalizes primitive value function over the Markov chain of policies.

## Meta-Multiagent Policy Gradient Theorem (Theorem 1)

**Statement:**

For any stochastic game M_n, the gradient of meta-value function for agent i w.r.t. φ^i_0 is:

```
∇_{φ^i_0} V^i_{φ_0:ℓ+1}(s_0, φ^i_0) = 

E_{p(τ_{φ_0:ℓ}|φ_0:ℓ)} E_{p(τ_{φ_ℓ+1}|φ_ℓ+1)} [G^i(τ_{φ_ℓ+1}) ×
  (∇_{φ^i_0} log π(τ_{φ_0}|φ^i_0) +                    [Current Policy]
   Σ_{ℓ'=0}^ℓ ∇_{φ^i_0} log π(τ_{φ_ℓ'+1}|φ^i_{ℓ'+1}) + [Own Learning]
   Σ_{ℓ'=0}^ℓ ∇_{φ^i_0} log π(τ_{φ_ℓ'+1}|φ^{-i}_{ℓ'+1})  [Peer Learning]
  )]
```

**Three components:**

1. **Current Policy:** Standard PG term for initial trajectory
2. **Own Learning:** How φ^i_0 affects agent's own future policies φ^i_1:L
   - Same as Meta-PG (Al-Shedivat 2018)
3. **Peer Learning:** How φ^i_0 affects other agents' future policies φ^{-i}_1:L
   - Similar to LOLA (Foerster 2018) but derived from first principles
   - Enables meta-agent to influence peers' learning dynamics

## Proof Sketch (Appendix A, for Ch.6)

### Step 1: Product Rule on Meta-Value Function

Start with:
```
V^i_{φ_0:ℓ+1}(s_0, φ^i_0) = E_{p(τ_{φ_0:ℓ}|φ_0:ℓ)} [Σ_{a^i_0} π(a^i_0|s_0, φ^i_{ℓ+1}) 
                                                    Σ_{a^{-i}_0} π(a^{-i}_0|s_0, φ^{-i}_{ℓ+1}) 
                                                    Q^i_{φ_ℓ+1}(s_0, a_0)]
```

Apply ∇_{φ^i_0} and product rule → 4 terms:
- **Term A:** ∇_{φ^i_0} p(τ_{φ_0:ℓ}|φ^i_{0:ℓ}, φ^{-i}_{0:ℓ})
- **Term B:** ∇_{φ^i_0} π(a^i_0|s_0, φ^i_{ℓ+1})
- **Term C:** ∇_{φ^i_0} π(a^{-i}_0|s_0, φ^{-i}_{ℓ+1})
- **Term D:** ∇_{φ^i_0} Q^i_{φ_ℓ+1}(s_0, a_0)

### Step 2: Expand Trajectory Probability Derivative (Term A)

Trajectory probability:
```
p(τ_{φ_ℓ}|φ^i_ℓ, φ^{-i}_ℓ) = p(s_0) Π_{t=0}^H π(a^i_t|s_t, φ^i_ℓ) π(a^{-i}_t|s_t, φ^{-i}_ℓ) P(s_{t+1}|s_t, a_t)
```

Chain of trajectories:
```
p(τ_{φ_0:ℓ}|φ^i_{0:ℓ}, φ^{-i}_{0:ℓ}) = p(τ_{φ_0}|φ^i_0, φ^{-i}_0) × ... × p(τ_{φ_ℓ}|φ^i_ℓ, φ^{-i}_ℓ)
```

Take gradient:
```
∇_{φ^i_0} [p(τ_{φ_0:ℓ})] = 
  Σ_{τ_{φ_0}} ∇_{φ^i_0} p(τ_{φ_0}|φ^i_0, φ^{-i}_0) Π_{ℓ'≠0} p(τ_{φ_ℓ'}|φ^i_{ℓ'}, φ^{-i}_{ℓ'}) +
  Σ_{τ_{φ_1}} ∇_{φ^i_1} p(τ_{φ_1}|φ^i_1, φ^{-i}_1) Π_{ℓ'≠1} p(τ_{φ_ℓ'}|φ^i_{ℓ'}, φ^{-i}_{ℓ'}) + ...
```

### Step 3: Log-Derivative Trick

Apply ∇ log x = ∇x / x:

```
∇_{φ^i_0} p(τ_{φ_ℓ}|φ^i_ℓ, φ^{-i}_ℓ) = p(τ_{φ_ℓ}) ∇_{φ^i_0} [log π(τ_{φ_ℓ}|φ^i_ℓ) + log π(τ_{φ_ℓ}|φ^{-i}_ℓ)]
```

**Crucial:** The log-terms include BOTH agent i's policy AND peers' policies!

This gives Term A in expectation form:
```
E[∇_{φ^i_0} log π(τ_{φ_0}|φ^i_0) + 
  Σ_{ℓ'=0}^{ℓ-1} ∇_{φ^i_0} log π(τ_{φ_ℓ'+1}|φ^i_{ℓ'+1}) +
  Σ_{ℓ'=0}^{ℓ-1} ∇_{φ^i_0} log π(τ_{φ_ℓ'+1}|φ^{-i}_{ℓ'+1})] × [...]
```

### Step 4: Unroll Q-Function Derivative (Terms B-D)

Repeatedly unroll ∇_{φ^i_0} Q^i_{φ_ℓ+1}(s_0, a_0) using Bellman equation (Sutton & Barto 1998):

```
∇_{φ^i_0} Q^i_{φ_ℓ+1}(s_0, a_0) = E[Σ_s ρ_{φ_ℓ+1}(s) Σ_a ∇_{φ^i_0} π^i(a^i|s, φ^i_{ℓ+1}) π^{-i}(a^{-i}|s, φ^{-i}_{ℓ+1}) Q^i(s,a)]
                                  + E[Σ_s ρ_{φ_ℓ+1}(s) Σ_a π^i(a^i|s, φ^i_{ℓ+1}) ∇_{φ^i_0} π^{-i}(a^{-i}|s, φ^{-i}_{ℓ+1}) Q^i(s,a)]
```

This adds consideration of future joint policy φ_ℓ+1.

### Step 5: Combine Everything

Terms A + B + C + D → final form:
```
∇_{φ^i_0} V^i_{φ_0:ℓ+1} = E[G^i(τ_{φ_ℓ+1}) × (Current Policy + Own Learning + Peer Learning)]
```

**QED**

## Connection to Prior Work

### Meta-PG (Al-Shedivat 2018)

**Assumption:** Other agents' learning is independent of meta-agent's behavior.

Mathematically: ∇_{φ^i_0} φ^{-i}_{ℓ'+1} = 0 for all ℓ'

**Result:** Peer learning term vanishes!

```
∇_{φ^i_0} V^i_{φ_0:ℓ+1} = E[G^i × (Current Policy + Own Learning)]
```

**Corollary (Remark 1):** Meta-PG is a special case of Meta-MAPG when peers' policies are treated as external factors.

### LOLA (Foerster 2018)

**Approach:** Use first-order Taylor approximation of value function to approximate influence on other agents' learning.

**Result:** Derives a term similar to peer learning gradient, but from different derivation.

**Meta-MAPG unifies both:**
- Own learning (from Meta-PG)
- Peer learning (similar to LOLA, but principled gradient derivation)

## Relevant Equations

**Meta-value function:**
```
V^i_{φ_0:ℓ+1}(s_0, φ^i_0) = E_{τ_{φ_0:ℓ}} E_{τ_{φ_ℓ+1}} [G^i(τ_{φ_ℓ+1})]
```

**Inner-loop update (policy gradient):**
```
φ^i_{ℓ+1} = φ^i_ℓ + α^i ∇_{φ^i_ℓ} E[G^i(τ_{φ_ℓ})]
φ^{-i}_{ℓ+1} = φ^{-i}_ℓ + α^{-i} ∇_{φ^{-i}_ℓ} E[G^{-i}(τ_{φ_ℓ})]
```

**Multiagent Policy Gradient (Wei et al. 2018):**
```
∇_{φ^i} E[G^i(τ_φ)] = Σ_s ρ_φ(s) Σ_a ∇_{φ^i} π^i(a^i|s,φ^i) π^{-i}(a^{-i}|s,φ^{-i}) Q^i_φ(s,a)
```

**Trajectory probability:**
```
p(τ_φ|φ^i, φ^{-i}) = p(s_0) Π_{t=0}^H π^i(a^i_t|s_t,φ^i) π^{-i}(a^{-i}_t|s_t,φ^{-i}) P(s_{t+1}|s_t,a_t)
```

**Meta-MAPG gradient (full):**
```
∇_{φ^i_0} V^i_{φ_0:ℓ+1} = E[G^i(τ_{φ_ℓ+1}) × (
  ∇_{φ^i_0} log π(τ_{φ_0}|φ^i_0) +
  Σ_{ℓ'=0}^ℓ ∇_{φ^i_0} log π(τ_{φ_ℓ'+1}|φ^i_{ℓ'+1}) +
  Σ_{ℓ'=0}^ℓ ∇_{φ^i_0} log π(τ_{φ_ℓ'+1}|φ^{-i}_{ℓ'+1})
)]
```

## Algorithm (simplified, for dissertation)

**Meta-training (outer loop):**
1. Sample initial peer policies φ^{-i}_0 ~ p(φ^{-i}_0)
2. For each peer:
   - Initialize φ^i_0 (meta-agent's initial policy)
   - For ℓ = 0 to L-1:
     - Collect trajectory τ_{φ_ℓ} ~ p(τ_{φ_ℓ}|φ_ℓ)
     - Inner-loop update: φ_{ℓ+1} = f(φ_ℓ, τ_{φ_ℓ}, α)
   - Compute meta-gradient ∇_{φ^i_0} Σ_ℓ V^i_{φ_0:ℓ+1}
3. Meta-update: φ^i_0 ← φ^i_0 + β ∇_{φ^i_0} Σ_ℓ V^i_{φ_0:ℓ+1}

**Meta-testing (deployment):**
1. Initialize φ^i_0 ← φ^i_0^* (optimized from meta-training)
2. Encounter new peer with hidden initial policy
3. Adapt using inner-loop updates (same as training)
4. Meta-learned φ^i_0^* enables fast adaptation!
